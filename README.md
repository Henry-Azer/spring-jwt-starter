# spring-security-jwt 
Spring security JWT starter, Spring boot App.  
  
•	APIs:  
    1- http://localhost:8080/api/auth/login [POST]  
    2- http://localhost:8080/api/user/info [GET]  
  
•	JSON Login Credentials  
    {  
        "email" : "email",  
        "password" : "password"  
    }
